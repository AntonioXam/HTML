Introducción

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Define la estructura y el contenido de una página web. CSS (Cascading Style Sheets) se utiliza para dar estilo a los elementos HTML, controlando aspectos como colores, fuentes y diseño.

Elementos básicos

Un elemento HTML está formado por una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<p>Este es un párrafo.</p>

Atributos

Los atributos proporcionan información adicional sobre un elemento. Se colocan en la etiqueta de apertura. Por ejemplo:

<a href="https://www.ejemplo.com">Enlace a un sitio web</a>

Encabezados

Hay seis niveles de encabezados, desde <h1> (el más importante) hasta <h6> (el menos importante):

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>

Párrafos

Los párrafos se definen con la etiqueta <p>:

<p>Este es un párrafo de texto.</p>

Enlaces

Los enlaces se crean con la etiqueta <a> y el atributo href:

<a href="https://www.ejemplo.com">Enlace a un sitio web</a>

Imágenes

Las imágenes se insertan con la etiqueta <img> y el atributo src:

<img src="imagen.jpg" alt="Descripción de la imagen">

Listas

Hay dos tipos de listas: ordenadas (<ol>) y desordenadas (<ul>):

<ol>
<li>Elemento 1</li>
<li>Elemento 2</li>
</ol>
<ul>
<li>Elemento A</li>
<li>Elemento B</li>
</ul>

Tablas

Las tablas se crean con la etiqueta <table>:

<table>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th>
</tr>
<tr>
<td>Dato 1</td>
<td>Dato 2</td>
</tr>
</table>

Formularios

Los formularios se crean con la etiqueta <form>:

<form action="/procesar_datos" method="post">
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">
<button type="submit">Enviar</button>
</form>

Estilos CSS

CSS se puede añadir de tres maneras:

  • En línea: usando el atributo style en la etiqueta HTML.
  • Interno: usando la etiqueta <style> en la sección <head> del documento HTML.
  • Externo: usando un archivo CSS separado y enlazándolo al documento HTML con la etiqueta <link>.

Selectores

Los selectores se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos:

  • Selector de etiqueta: selecciona todos los elementos de un tipo específico (p. ej., p, h1, div).
  • Selector de clase: selecciona todos los elementos con una clase específica (p. ej., .destacado).
  • Selector de ID: selecciona un elemento específico con un ID único (p. ej., #menu).

Propiedades

Las propiedades definen los estilos que se aplicarán a los elementos seleccionados. Algunas propiedades comunes son:

  • color: define el color del texto.
  • font-size: define el tamaño de la fuente.
  • background-color: define el color de fondo.
  • margin: define el margen alrededor de un elemento.
  • padding: define el espacio interno de un elemento.

Ejemplo de CSS:

p {
color: blue;
font-size: 16px;
}
