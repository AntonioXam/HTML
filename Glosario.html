Glosario de HTML5 con ejemplos

Este glosario incluye los componentes más comunes de HTML5 con una breve descripción y un ejemplo práctico para que puedas copiar y pegar en tu código.

  • Estructura de la página
  • Etiquetas de sección
  • Etiquetas de contenido
  • Etiquetas multimedia
  • Etiquetas de enlace
  • Etiquetas de incrustación
  • Etiquetas semánticas

Estructura de la página

<!DOCTYPE html>
Define el tipo de documento como HTML5. <!DOCTYPE html>
<html lang="es">
Etiqueta raíz del documento. El atributo lang especifica el idioma. <html lang="es"> <head> ... </head> <body> ... </body> </html>
<head>
Contiene metadatos sobre el documento. <head> <meta charset="UTF-8"> <title>Mi página web</title> </head>
<title>
Define el título que aparece en la pestaña del navegador. <title>Mi página web</title>
<meta charset="UTF-8">
Define la codificación de caracteres. <meta charset="UTF-8">
<body>
Contiene el contenido visible de la página. <body> <h1>Hola mundo</h1> <p>Este es un párrafo de texto.</p> </body>

Etiquetas de sección

<header>
Define la cabecera de una sección o página. <header> <h1>Mi sitio web</h1> <nav> ... </nav> </header>
<nav>
Define un conjunto de enlaces de navegación. <nav> <ul> <li><a href="#">Inicio</a></li> <li><a href="#">Acerca de</a></li> <li><a href="#">Contacto</a></li> </ul> </nav>
<main>
Define el contenido principal de la página. <main> <article> ... </article> <aside> ... </aside> </main>
<article>
Define una sección independiente de contenido. <article> <h2>Título del artículo</h2> <p>Contenido del artículo.</p> </article>
<section>
Define una sección genérica dentro de un documento. <section> <h2>Sección 1</h2> <p>Contenido de la sección 1.</p> </section>
<aside>
Define contenido relacionado tangencialmente con el contenido que lo rodea. <aside> <h3>Información adicional</h3> <p>Contenido relacionado.</p> </aside>
<footer>
Define el pie de página de una sección o página. <footer> <p>Copyright 2024</p> </footer>

Etiquetas de contenido

<h1> a <h6>
Define encabezados de diferentes niveles. <h1>Encabezado principal</h1> <h2>Encabezado secundario</h2>
<p>
Define un párrafo de texto. <p>Este es un párrafo de texto.</p>
<br>
Inserta un salto de línea. <p>Esta es la primera línea.<br>Esta es la segunda línea.</p>
<hr>
Inserta una línea horizontal. <p>Texto arriba de la línea.</p> <hr> <p>Texto debajo de la línea.</p>
<pre>
Define texto preformateado. <pre> Este texto se muestra exactamente como se escribe. </pre>
<blockquote>
Define una cita larga. <blockquote> <p>Esta es una cita larga.</p> </blockquote>
<q>
Define una cita corta. <p>Dijo <q>Hola</q> al entrar.</p>
<ul>
Define una lista desordenada. <ul> <li>Elemento 1</li> <li>Elemento 2</li> <li>Elemento 3</li> </ul>
<ol>
Define una lista ordenada. <ol> <li>Primer elemento</li> <li>Segundo elemento</li> <li>Tercer elemento</li> </ol>
<li>
Define un elemento de lista. <li>Elemento de lista</li>
<dl>
Define una lista de definiciones. <dl> <dt>HTML</dt> <dd>Lenguaje de marcado de hipertexto.</dd> <dt>CSS</dt> <dd>Hojas de estilo en cascada.</dd> </dl>
<dt>
Define un término en una lista de definiciones. <dt>HTML</dt>
<dd>
Define la definición de un término. <dd>Lenguaje de marcado de hipertexto.</dd>
<table>
Define una tabla. <table> <tr> <th>Nombre</th> <th>Apellido</th> </tr> <tr> <td>Juan</td> <td>Pérez</td> </tr> </table>
<tr>
Define una fila en una tabla. <tr> <td>Juan</td> <td>Pérez</td> </tr>
<td>
Define una celda en una tabla. <td>Juan</td>
<th>
Define una celda de encabezado en una tabla. <th>Nombre</th>
<form>
Define un formulario. <form action="/enviar" method="post"> <label for="nombre">Nombre:</label> <input type="text" id="nombre" name="nombre"> <button type="submit">Enviar</button> </form>
<input>
Define un campo de entrada en un formulario. <input type="text" id="nombre" name="nombre">
<textarea>
Define un área de texto en un formulario. <textarea id="mensaje" name="mensaje"></textarea>
<button>
Define un botón. <button type="submit">Enviar</button>
<select>
Define una lista desplegable. <select id="pais" name="pais"> <option value="españa">España</option> <option value="mexico">México</option> <option value="argentina">Argentina</option> </select>
<option>
Define una opción en una lista desplegable. <option value="españa">España</option>
<label>
Define una etiqueta para un elemento de formulario. <label for="nombre">Nombre:</label>
<fieldset>
Agrupa elementos relacionados en un formulario. <fieldset> <legend>Datos personales</legend> <label for="nombre">Nombre:</label> <input type="text" id="nombre" name="nombre"> </fieldset>
<legend>
Define un título para un <fieldset>. <legend>Datos personales</legend>

Etiquetas multimedia

<img>
Inserta una imagen. <img src="imagen.jpg" alt="Descripción de la imagen">
<audio>
Inserta un archivo de audio. <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>
<
Inserta un archivo de vídeo. <video controls> <source src="video.mp4" type="video/mp4"> </video>

Etiquetas de enlace

<a>
Define un hipervínculo. <a href="https://www.ejemplo.com">Enlace a un sitio web</a>

Etiquetas de incrustación

<iframe>
Inserta otro documento HTML dentro del actual. <iframe src="https://www.ejemplo.com"></iframe>

Etiquetas semánticas

<figure>
Define una figura (ej. una imagen con su pie de foto). <figure> <img src="imagen.jpg" alt="Descripción"> <figcaption>Pie de foto</figcaption> </figure>
<figcaption>
Define el pie de foto de una figura. <figcaption>Pie de foto</figcaption>
<details>
Define detalles adicionales que el usuario puede mostrar u ocultar. <details> <summary>Detalles</summary> <p>Información adicional</p> </details>
<summary>
Define un encabezado visible para <details>. <summary>Detalles</summary>
<time>
Define una fecha y/o hora. <time datetime="2024-10-24">24 de octubre de 2024</time>
<mark>
Resalta texto. <p>Este texto está <mark>resaltado</mark>.</p>
