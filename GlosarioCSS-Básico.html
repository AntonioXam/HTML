Sintaxis básica de CSS

Una regla CSS tiene esta estructura:

selector {
  propiedad: valor;
  propiedad: valor;
}

¿Qué significa cada parte?

  • selector: Indica a qué elemento HTML se aplicará el estilo.
  • propiedad: Es el atributo que queremos modificar (color, tamaño...).
  • valor: Es el valor que le damos a la propiedad.

Ejemplo:

p {
  color: red;
  font-size: 18px;
}

Este código CSS hará que todos los párrafos (<p>) tengan texto rojo y un tamaño de letra de 18 píxeles.

Selectores

Los selectores nos permiten elegir qué elementos HTML queremos estilizar.

Tipos básicos:

  • Selector de etiqueta: Selecciona por el nombre de la etiqueta (p, h1, div...).
    h1 { color: blue; }
  • Selector de clase: Selecciona elementos con un atributo de clase específico (.mi-clase).
    .resaltado { font-weight: bold; }
  • Selector de ID: Selecciona un único elemento con un ID específico (#mi-elemento).
    #titulo-principal { font-size: 36px; }

Colores

Podemos especificar colores de varias maneras:

  • Por nombre: red, blue, green, black, white...
    p { color: green; }
  • Código hexadecimal: #RRGGBB (ej. #FF0000 para rojo).
    h2 { color: #00FF00; } /* Verde */

Propiedades de texto

Podemos cambiar la apariencia del texto:

  • color: Cambia el color del texto.
    p { color: purple; }
  • font-family: Cambia el tipo de letra.
    h1 { font-family: Arial, sans-serif; }
  • font-size: Cambia el tamaño de la letra.
    p { font-size: 16px; }
  • font-weight: Cambia el grosor de la letra (normal, bold...).
    strong { font-weight: bold; }
  • text-align: Alinea el texto (left, center, right...).
    p { text-align: center; }

Propiedades de fondo

Podemos cambiar el fondo de los elementos:

  • background-color: Cambia el color de fondo.
    div { background-color: lightblue; }

Propiedades de bordes

Podemos añadir bordes a los elementos:

  • border: Define el borde (grosor, estilo y color).
    p { border: 2px solid black; }

Modelo de caja

Cada elemento HTML es como una caja con estas partes:

  • Contenido: Lo que hay dentro del elemento (texto, imágenes...).
  • Padding: Espacio entre el contenido y el borde.
  • Borde: Lalínea que rodea al elemento.
  • Margen: Espacio fuera del elemento, que lo separa de otros.

Propiedades importantes:

  • width y height: Definen el ancho y alto del contenido.
    div { width: 200px; height: 100px; }
  • padding: Define el padding (espacio interno).
    p { padding: 10px; }
  • margin: Define el margen (espacio externo).
    img { margin: 20px; }
